# 0004. R 実行環境の選択

## Status

Accepted

## Context

ブラウザ上で R コードを実行できる環境が必要。
サーバーレスで動作する必要がある（[ref](./0002-ホスティングプラットフォームの選択.md)）。

## Decision

[WebR](https://docs.r-wasm.org/webr/latest/) を選択する

## Consequences

- **メリット**
  - ブラウザ内で R を実行可能
  - サーバーサイドの処理が不要
  - 追加の環境構築が不要
- **デメリット**
  - 比較的新しい技術で実装例が少ない
  - 初期ロード時間がかかる可能性
  - ブラウザ互換性の確認が必要
