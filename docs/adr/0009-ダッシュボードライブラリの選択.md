# 0009. ダッシュボードライブラリの選択

## Status

Accepted

## Context

インタラクティブなダッシュボードを提供する必要がある。Next.js（React）アプリケーションで使用する。

## Decision

[Recharts](https://recharts.github.io/en-US/examples/) を選択する。

## Consequences

- **メリット**
  - Reactとの統合が容易
  - シンプルな API で使いやすい
  - カスタマイズ性が高い
  - 軽量
- **デメリット**
  - 複雑なデータ可視化には不向きな場合がある
  - 大量のデータポイントではパフォーマンスが低下する可能性
