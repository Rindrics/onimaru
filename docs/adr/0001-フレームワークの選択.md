# 0001. フレームワークの選択

## Status

Accepted

## Context

ブラウザ上で動作する Web アプリケーションを構築する必要がある。
提供する機能は下記:
- 検索・フィルタ UI
- ダッシュボード
- オンライン分析環境
- 静的ページ

## Decision

Next.jsを選択する

## Consequences

- **メリット**
  - Reactベースで豊富なエコシステム
  - サーバーサイドレンダリング（SSR）と静的サイト生成（SSG）をサポート
  - Vercelとの統合が容易
  - TypeScriptのサポートが充実
- **デメリット**
  - 学習曲線がある（Reactの知識が必要）
