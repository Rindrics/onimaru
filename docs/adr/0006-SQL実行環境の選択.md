# 0006. SQL 実行環境の選択

## Status

Accepted

## Context

ブラウザ上で SQL クエリを実行できる環境が必要。
Parquet ファイルに対してクエリを実行する必要がある（[ref](./0003-データ形式の選択.md)）。
実装順序として最後にSQL環境を提供したい。

## Decision

DuckDB-WASM を選択する。

## Consequences

- **メリット**
  - ブラウザ内で SQL クエリを実行可能
  - Parquet ファイルを直接読み込める
  - 軽量で高速
  - サーバーサイドの処理が不要
- **デメリット**
  - ブラウザのリソース制限に依存
  - 大規模なデータセットではパフォーマンスが低下する可能性
