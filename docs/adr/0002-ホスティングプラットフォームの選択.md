# 0002. ホスティングプラットフォームの選択

## Status

Accepted

## Context

本サービスは無償教育を目的としているため、インフラ料金をなるべく安価に抑えたい
また、Next.js アプリケーションをデプロイする必要がある（[ref](./0001-フレームワークの選択.md)）。

## Decision

Vercel を無料プランで利用する

## Consequences

- **メリット**
  - 無料で運用可能
  - Next.jsとの親和性が高く、デプロイが容易
  - グローバル CDN による高速配信
  - 自動デプロイ機能
- **デメリット**
  - 無料プランには実行時間やリクエスト数の制限がある
  - サーバーレス関数の実行時間制限（10秒）
