# Onimaru

東京大学大気海洋研究所が提供するオープンデータ "[大槌湾観測データベース](https://opensci.aori.u-tokyo.ac.jp/otsuchi.html)" を再配布するサービスです。

## 提供する価値

本サービスは、既存の配信サービスに下記の機能を追加して再配布します：

1. 充実した検索・フィルタ機能
   - 多様な条件でデータを検索・フィルタリングできる
   - 効率的に目的のデータを発見できる
2. ブラウザ上でのオンライン分析環境
   - R、Python、SQLをブラウザ上で実行できる環境を提供
   - 環境構築不要で、すぐにデータ分析を開始できる
3. インタラクティブなダッシュボード
   - 提供するデータの概要を直感的に理解できる
4. 学習コンテンツ
   - 海洋学の基礎を実際のデータを使って学べる

これによって、下記のような成果が得られることを狙っています:

- 大槌湾データの利用促進
- 海洋学・データ操作スキルについての普及教育
- ウェブサービス開発手法についての教育
- 学術研究のアウトリーチ形態の設計事例としての成果発信
